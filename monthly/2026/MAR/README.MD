# Readings

## The Pragmatic Programmer (ISBN: 978-0-13-595705-9)

Chapter 10: Postface

- As software engineers we have a privilege of shaping the future. It can be optimistic or dystopic.
- With power comes responsibility: automatic heating software can even kill someone. Non-embedded software may cause
  harm as well: banking systems handle doubts which influence other people's lives.
- Ask yourself two questions:
  - Would you like to be a user of your software?
  - What did you do to protect users?
